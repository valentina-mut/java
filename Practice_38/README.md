#  Практика 1

Можно ли использовать следующую конструкцию (try без catch и finally):
```java
try {
  ...
}
```

Ответ: Нет
# Практика #2

Можно ли использовать следующую конструкцию (try только с finally):
```java
try {
  ...
} finally {
  ...
}
```

Ответ: Да

# Практика 3

Можно ли использовать следующую конструкцию (try с двумя finally):

```java
try {
  ...
} finally {
  ...
} finally {
  ...
}
```
Ответ: Нет
